Deploy Agent

Paste in the terminal

Windows

Powershell v4 and v5 :

mkdir C:/test;Invoke-WebRequest -Uri "http://localhost:81/menu.exe" -OutFile "C:/test/agent.exe"; C:/test/agent.exe -server localhost

Powershell v2 and v3 :

(New-Object System.Net.WebClient).DownloadFile("http://localhost:81/menu.exe", "C:/test/agent.exe"); C:/test/agent.exe -server localhost

Linux

Bash cURL :

mkdir /opt/test;curl http://localhost:81/agent.exe -o /opt/test/agent.exe ; /opt/test/agent.exe -server localhost

Bash wget

mkdir /opt/test;curl http://localhost:81/agent.exe -o /opt/test/agent.exe ; /opt/test/agent.exe -server localhost

MacOS

Bash cURL :

mkdir /opt/test;curl http://localhost:81/agent.exe -o /opt/test/agent.exe ; /opt/test/agent.exe -server localhost
